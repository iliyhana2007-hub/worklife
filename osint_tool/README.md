# Telegram OSINT Scanner

Этот инструмент позволяет искать информацию о пользователе Telegram и проверять его наличие в группах, используя ваш собственный аккаунт.

## Установка

1. Убедитесь, что установлен Python 3.
2. Установите зависимости:
   ```bash
   pip3 install -r requirements.txt
   ```

## Настройка

Для работы скрипта требуются `API_ID` и `API_HASH` от Telegram (это бесплатно).

1. Перейдите на https://my.telegram.org/apps
2. Авторизуйтесь под своим аккаунтом.
3. Создайте любое приложение (название и описание могут быть любыми).
4. Скопируйте `App api_id` и `App api_hash`.
5. Вы можете вставить их в файл `.env.example` (и переименовать его в `.env`) ИЛИ просто ввести при запуске скрипта.

## Использование

1. Запустите скрипт:
   ```bash
   python3 main.py
   ```
2. Введите номер телефона (для первого входа) и код подтверждения от Telegram.
3. Введите username цели (например `@durov`).
4. Скрипт покажет:
   - ID, Био, Фото.
   - **Common Chats**: Чаты, где вы оба состоите.
   - Предложит запустить **Сканер**.

## Как работает Сканер (Scanner)

Скрипт берет список групп из файла `target_chats.txt` и проверяет каждую группу: состоит ли там целевой пользователь.
- По умолчанию в `target_chats.txt` добавлено несколько популярных IT-чатов.
- **Чтобы найти больше**, вам нужно добавить в этот файл ссылки на группы (по одной на строке), в которых вы хотите искать человека (например, городские чаты, чаты по интересам и т.д.).

## ВАЖНО

- Скрипт делает паузу 1.5 секунды между запросами, чтобы избежать бана (FloodWait).
- Не уменьшайте задержку, если не уверены.
- Это работает только с ПУБЛИЧНЫМИ чатами.
